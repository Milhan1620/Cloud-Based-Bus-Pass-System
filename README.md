# 🚌 Cloud-Based Bus Pass System

A cloud-based bus ticket booking and digital bus pass management system developed using **Flask**, **SQLite**, and **Bootstrap** as part of the **Cloud Computing Internship - Task 02**.


## 📌 Project Overview

This project provides an online platform where passengers can register, log in, search available routes, book bus seats, and download QR-based digital tickets.

The system is designed to reduce ticket loss, prevent duplicate bookings, and improve reliability using cloud computing concepts.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YourUsername/Cloud-Bus-Pass-System-Task02.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
